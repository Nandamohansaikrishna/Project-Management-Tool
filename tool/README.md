# Project Management

A React-based web application with a sign-in page and dashboard, built with Vite and styled using Bootstrap.

## Setup
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Open `http://localhost:5173` in your browser.

## Features
- Sign-in page with form validation
- Dashboard with key sections: Team Members, Projects, Tasks
- Animations and Bootstrap styling
